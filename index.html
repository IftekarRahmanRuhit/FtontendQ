HTML

HTML(Hypertext Markup Language) related interview questions

Doctype

Question: What is Doctype? Why do you need it?

Answer: Doctype is an instruction to the browser to inform about the version of html document and how browser should render it. It ensures how element should be displayed on the page by most of the browser. And it also makes browser's life easier. otherwise, browser will guess and will go to quirks mode. Moreover, doctype is required to validate markup.

Data Attribute

Question: What is the use of Data-Attribute?

Answer: Allow you to store extra information/ data in the DOM. You can write valid html with embedded private data. You can easily access data attribute by using javascript and hence a lot of libraries like knockout uses it.

Mark

Question: How can you highlight text in html?

Answer: using mark element.

Scoped

Question: Can you apply css rule to a part of html document?

Answer: Yes, i can apply CSS rules to specific parts of an HTML document by targeting the elements i want to style.
